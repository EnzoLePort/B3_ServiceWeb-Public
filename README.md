# B3_ServiceWeb-Public
# ======
# Notes
Au clic du bouton en haut à droite on va charger différntes données de l'API themoviedb<br/>
Au clic du bouton like : +1 like (Bug : aucune limite / mise à jour du compte au clic uniquement)<br/>
Au clic d'une image d'un film : affichage de la bande d'annonce ! (Si celle-ci est disponible)<br/>
# NB :
Pour visualiser toutes les donnees, ne pas oublier de rentrer son api_key dans le fichier .config.js !<br/>
